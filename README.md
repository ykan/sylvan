# 环境要求

- node ^22
- pnpm ^8

# 开发脚本

```bash
~ pnpm dev    # 启动开发环境
~ pnpm commit # 提交代码
~ pnpm format # 格式化代码
~ pnpm lint   # 检查代码
```

## 拉取环境变量

```bash
~ pnpm i -g vercel
~ vercel login
~ vercel link
~ vercel env pull
```
